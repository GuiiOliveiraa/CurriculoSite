Desenvolvi a Nunes Store, uma loja virtual completa de produtos.

Features:
- Interface responsiva com filtros dinâmicos
- Carrinho de compras com localStorage
- Integração com WhatsApp para conversão (envia carrinho para cliente via link)
- Otimização de imagens para SEO
- Design moderno sem frameworks (vanilla HTML/CSS/JS)

Stack: HTML5, CSS3, JavaScript vanilla, Font Awesome Icons, WhatsApp Cloud API

O projeto demonstra:
- Desenvolvimento frontend puro (sem React/Vue)
- Manipulação dinâmica do DOM
- Consumo de dados via JSON estático
- Integração com APIs externas (WhatsApp)
- Boas práticas de responsividade e performance
