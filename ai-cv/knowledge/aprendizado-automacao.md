Aprendizagem em automação e processos:

Durante o desenvolvimento do CurriculoSite, aprendi sobre:
- GitHub Actions e pipelines de CI/CD
- Processamento automático de arquivos com hashes para evitar reprocessamento
- Integração com LLMs (Gemini, arquitetura para múltiplos provedores)
- Tratamento de rate limits e estratégias de retry
- Logging em produção

Este projeto é um exemplo real de como aplicar automação para manter dados atualizados
sem intervenção manual, de forma escalável e profissional.
